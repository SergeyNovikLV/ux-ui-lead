# Client logos (Worked with)

Place PNG logo files here. Filenames must be:

| File       | Brand              |
|-----------|--------------------|
| `c4.png`  | Channel 4          |
| `db.png`  | Deutsche Bank      |
| `jj.png`  | Johnson & Johnson  |
| `pm.png`  | Perfetti Van Melle |
| `mapon.png` | Mapon            |

- Use **grayscale** PNGs (or color; they are shown in grayscale by default).
- On hover, logos switch to full color.
- Recommended height: ~100–120px in the image so they scale cleanly to 36px display height.
